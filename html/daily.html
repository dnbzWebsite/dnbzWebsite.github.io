如何显示文件扩展名？

Win7

开始 -> 文件夹选项 -> 查看 -> 隐藏已知文件扩展名 -> 关闭

Win10

此电脑 -> 查看 -> 文件扩展名 -> 开启

如何正确卸载软件？

控制面板 -> 程序和功能 -> 选择想卸载的软件并双击进入卸载程序

有必要更改电源选项吗？

不要无脑选高性能/卓越模式，其本质是让CPU锁频，对于游戏帧数的稳定性有帮助，但不能提高性能，玩网游（CSGO，吃鸡）等游戏的用户可以按情况打开，选择平衡模式即可，笔记本用户可选择厂商定制的电源模式

有必要更改N卡控制面板的模式吗？

没必要，NV的驱动功能较老，建议在游戏内设置

有必要更改虚拟内存吗？

没必要，其速度无法跟内存相比，只是用来存放活动数据，且系统默认会设置好

有必要安装杀毒软件吗？

如果是小白推荐安装360，火绒等杀毒软件，如果你具有辨别病毒的能力，可以选择不装

有什么推荐的杀毒软件吗？

国产：360，火绒，智量

国外：卡巴斯基，小红伞

如何清理系统缓存？

Windows键（徽标键） + R -> 运行 -> 输入“%temp%” -> 进入缓存目录并将文件全部删除，如出现无法删除的请选择跳过

如何关闭某些开机启动项？

Windows键（徽标键） + R -> 运行 -> 输入“msconfig” -> 启动 -> 选择要关闭的启动项并关闭

如何校准时间？

控制面板 -> 日期和时间 -> Internet时间 -> 更改设置 -> 立即更新 -> 确定
